# Fuelle Roadmap

## v0.5.0 — Alpha ✅
- Core meal logging
- USDA food search
- Daily/weekly/history views
- Export/Import backup code
- Goals setting

## v1.0.0 — Stable
- Fat macro displayed in daily summary
- Manual food entry (no search needed)
- Copy previous day's meals as template
- Improved search (portion units beyond grams)
- Better offline support (top 500 foods bundled)
- App icons for all platforms

## v1.1.0
- Grocery list generation from meal plan
- Grocery list export in Flowe-compatible format (cross-app via user)
- Weekly meal planning (set future days in advance)

## v2.0.0
- Native iOS app (HealthKit integration)
- Native Android app (Health Connect integration)
- Barcode scanning for packaged foods
- Recipe builder (combine ingredients into one entry)
- Charts: calorie trend, macro breakdown over time

## Future
- Multi-profile support (family members)
- App Store & Google Play releases
